About the role:

We’re looking for a Senior Software Engineer – AI to build AI Agents for QMetry Test Management (QTM). You’ll work on LLM-powered, agentic workflows that automate and enhance software testing processes.

  • Build and deploy AI Agents for test management workflows
  • Develop LLM-based and MCP-style integrations
  • Prototype and validate new AI use cases
  • Work across backend APIs and product integrations
  • Deliver high-quality, production-ready code

Requirements:

  • 5+ years experience in Python, Node, JS,
  • Familiarity with Flask/FastAPI
  • Exposure to LLMs / LangChain / Agent frameworks
  • Strong understanding of REST APIs & system design basics
  • Experience with Docker, AWS, GitHub
  • Bonus: NoSQL / Vector DB / basic frontend (JS/TS)
